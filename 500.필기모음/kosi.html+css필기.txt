nth-child와 nth-of-type의 차이
- nth-child : 속성 상관 없이 순서대로 설정하는 것
- nth-of-type : 속성끼리 순서대로 설정하는 것

outline과 border의 차이
- outline : 주변에 영향 X (보이기만 함)
- border : 주변에 영향 O (영역에 영향을 미침)

CSS에서 함수를 지정할 때 (띄어쓰기)와 >의 차이
- (띄어쓰기) : 하위 모든 자손 중 하나 지정
- > : 한 단계 아래 자손 중 하나 지정
